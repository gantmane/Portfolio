## Areas

Use these pages when you want to quickly explain a domain in an interview and then jump straight to the implementation folders in the repo.

- **Cloud Security**: multi-account AWS, IAM, KMS, GuardDuty/Security Hub, VPC security, WAF/Shield, Zero Trust.
- **DevSecOps**: CI/CD security gates, SAST/DAST/SCA, secrets/IaC/container scanning, supply chain, policy as code.
- **Kubernetes Security**: EKS hardening, PSS, network policies, Istio mTLS/authz, Falco, Gatekeeper.
- **SIEM & SOC**: Wazuh + Security Onion, custom detections, IR playbooks, threat intel, MITRE mapping.
- **PCI DSS 4.0**: control mapping + automation patterns (cloud-native).

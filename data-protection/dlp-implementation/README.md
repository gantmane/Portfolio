## Overview
Enterprise Data Loss Prevention using AWS Macie, scanning 500TB of data, detecting 85,000+ PII/PHI instances, and blocking 3,200+ policy violations annually.

## Key Metrics
- **Data Scanned**: 500TB (S3, databases, endpoints)
- **PII/PHI Detected**: 85,000+ instances
- **Policy Violations Blocked**: 3,200+ annually
- **Data Exfiltration Attempts**: 120+ blocked
- **Email DLP Blocks**: 1,800+ annually
- **False Positive Rate**: <8%

## DLP Controls
1. **AWS Macie**: S3 bucket scanning for PII/PHI
2. **Email Gateway**: Block sensitive data in emails
3. **Endpoint DLP**: Prevent USB/cloud uploads
4. **Database Activity Monitoring**: Track sensitive queries

## Technology Stack
- AWS Macie
- Proofpoint Email DLP
- CrowdStrike Endpoint DLP
- Imperva Database Activity Monitoring
